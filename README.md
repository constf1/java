# java
small java projects

## Freecell Solver
### Compile
```shell
javac freecell\FreecellSolution.java
```
### Run
```shell
java freecell.FreecellSolution --deal 1
```
### JAR File
```shell
jar cvfe freecell-solver.jar freecell.FreecellSolution  freecell\*.class common\*.class
java -jar freecell-solver.jar --deal 1
```
